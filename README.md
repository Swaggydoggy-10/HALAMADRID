
# Universal Bank - Personal Loan Propensity Dashboard

Streamlit Cloud app for the Head of Marketing at XYZ Bank to:
- Explore customer insights
- Compare Decision Tree, Random Forest, and Gradient Boosting models (with 5-fold CV)
- Upload new customer data and predict Personal Loan propensity

## How to deploy

1. Create a GitHub repository.
2. Add these files in the ROOT of the repo (no folders):
   - `UniversalBank.csv`
   - `app.py`
   - `requirements.txt`
   - `README.md`
3. Go to Streamlit Community Cloud, create a new app, and point it to `app.py`.
4. Run the app.
